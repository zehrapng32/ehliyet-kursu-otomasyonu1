# Ehliyet Kursu Otomasyonu

Projenin amacı, bir ehliyet kursunda bulunan kursiyerlerin temel bilgilerinin
bilgisayar ortamında yönetilmesini sağlamak.

## Proje İçeriği

Bu projede aşağıdaki işlemler gerçekleştirilebilmektedir:
- Kursiyer ekleme
- Kursiyer silme
- Kursiyer arama
- Kursiyer listeleme
- Ehliyet sınıfları hakkında bilgilendirme

## Kullanılan Teknolojiler
- Programlama dili: C++
- Geliştirme ortamı: Visual Studio
- Versiyon kontrol sistemi: Git & GitHub

## Çalışma Şekli
Program konsol tabanlıdır. Kullanıcı menü üzerinden seçim yaparak
istenilen işlemleri gerçekleştirebilmektedir.
